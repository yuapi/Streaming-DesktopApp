## 기능
- 치지직 웹 사이트 이용
- 디스코드 활동 표시 기능
  - 시청중인 채널, 방송제목, 썸네일, 업타임 표시 (생방송)
  - 시청중인 채널, ("[다시보기]") 방송제목, 썸네일 표시 (VOD)
